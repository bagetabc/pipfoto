# PIPFOTO
🎨 PipFoto Tool

PipFoto Tool — это легковесный консольный инструмент для конвертации изображений в бинарный формат .pipfoto и их отображения прямо в терминале с использованием ANSI-цветов.

🚀 Возможности

Быстрое кодирование: Превратите любую картинку (JPG/PNG) в компактный формат .pipfoto.

Цветной вывод: Использование 24-битных цветов для максимально точной передачи оригинала в консоли.

Портативность: Легкий скрипт на Python, который работает на любой ОС.

🛠 Установка

Убедитесь, что у вас установлен Python, затем установите библиотеку Pillow:

`pip install Pillow`


📖 Использование

Кодирование изображения

Для преобразования картинки в формат .pipfoto используйте команду:

`python pipfoto_tool.py encode input.jpg output.pipfoto`


Декодирование (просмотр)

Для вывода картинки прямо в консоль:

`python pipfoto_tool.py decode output.pipfoto`


Совет для Windows: Чтобы открывать картинки простым перетаскиванием, создайте файл RUN.bat с содержимым:

`@echo off
chcp 65001 >nul
python "%~dp0pipfoto_tool.py" decode "%~1"
pause`


Теперь просто перетащите любой файл .pipfoto на этот .bat файл.
